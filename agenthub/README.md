# Agent Resources

This repository contains shared agents, skills, and prompts for your organization.

## Structure

- `.agents/` - AI Agent definitions (files named *.agent.md)
- `.skills/` - Reusable skills (folders containing SKILL.md)
- `.prompts/` - Reusable prompt templates (files named *.prompt.md)

## Usage

1. Add your agents, skills, and prompts to the respective folders
2. Use the VS Code "Agent Resources" extension to browse and import resources
3. Share this repository with your team

## File Naming Conventions

- **Agents**: `.agents/<name>.agent.md`
- **Skills**: `.skills/<skill-name>/SKILL.md`
- **Prompts**: `.prompts/<name>.prompt.md`

## Contributing

1. Create files following the naming conventions above
2. Include documentation and examples in your files
3. Commit and push your changes
